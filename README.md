### Run Locally

To run **animechill** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/DVHcoding/animechill.git
```

Windows:

```bash
git clone https://github.com/DVHcoding/animechill.git
```
